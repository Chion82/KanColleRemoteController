KanColle Remote Controller

Automate expeditions and sorties, view your ship and material status from remote.

Convenient

No spare time for gaming? You can now control your game even when you are busy and miles away from home using your smart phone browser. The project helps you proceed routine expeditions and sorties. Gain EXP and earn materials at work!

Easy

KanColle Remote Controller is easy to set up, use and maintain. The project has been tested on Windows 8.1, Windows Server 2012 with perfect compatibility. Refer to the KanColleRemoteController project on Github to get started!

Reliable

Please be aware that KanColle Remote Controller is NOT a cheating or hacking program but a game helper for busy KanColle players. It simulates mouse actions and captures real-time game data from the open-source project KanColleViewer.
